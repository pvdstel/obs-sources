OBS browser sources

  • Timer: add to OBS with the Shutdown source when not visible option to restart the timer when it becomes visible.

    Options:

    • name: name of the timer
    • start: start time in seconds
    • step: second increase per second (defaults to -1)
    • background: background color (defaults to transparent)
    • color: text color (defaults to black)
    • fontFamily: font family for the text
    • fontSize: font size for the text (defaults to 32px)
    • fontWeight: font weight for the text (defaults to bold)
    • padding: spacing around the text
    • ring: controls whether a notification sound is played when the timer reaches zero (off by default)
    • ringtone: the audio file that is played for the notification
